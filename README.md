
# minjur

Osmium-based converter of OSM data to GeoJSON.


## Prerequisites

You need [http://github.com/osmcode/libosmium](libosmium) installed.


## Compile

To setup:

    mkdir build
    cd build
    cmake ..

If needed configure using

    ccmake .

Compile with

    make


## Run

Call with `--help` to see command line options.
